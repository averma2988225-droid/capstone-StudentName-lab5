# DEVAN CLOTHING - E-commerce Website

## Project Title & Theme
**DEVAN CLOTHING** - E-commerce Store UI  
A fully responsive fashion retail website featuring product listings, filter sidebar, testimonials, and subscription form.

## Sections Implemented
- **Header/Navigation**: Sticky header with logo, navigation menu, and login button
- **Hero Banner**: Animated banner with call-to-action button
- **Flash Sale Banner**: Promotional banner with pulse animation
- **Product Grid**: 6 clothing items in responsive grid layout
- **Filter Sidebar**: Category, price, and color filters
- **Testimonials**: Customer reviews section
- **Newsletter Subscription**: Email signup form
- **Footer**: Multi-column footer with links and social icons

## Features Implemented

### Core Sections
- **Header/Navigation**: Logo, main navigation menu, login button with sticky positioning
- **Hero Banner**: Eye-catching banner with call-to-action and fade-in animation
- **Flash Sale Banner**: Animated promotional banner with pulse effect
- **Product Grid**: Responsive grid layout showcasing clothing items with hover effects
- **Filter Sidebar**: Category, price, and color filters for product browsing
- **Testimonials**: Customer reviews in responsive grid layout
- **Newsletter Subscription**: Email signup form with validation
- **Footer**: Multi-column footer with links, social icons, and company info

### Technical Implementation
- **Semantic HTML5**: Proper use of `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`
- **CSS Grid & Flexbox**: Modern layout techniques for responsive design
- **CSS Variables**: Consistent color scheme and spacing using custom properties
- **Animations**: Keyframe animations for hero section and flash banner
- **Hover Effects**: Interactive elements with smooth transitions
- **Accessibility**: Skip links, ARIA labels, proper form labels, screen reader support

## Breakpoints Used & Main Layout Techniques

### Responsive Breakpoints
- **Desktop (≥1200px)**: 3-column product grid, horizontal navigation
- **Tablet (≤768px)**: Stacked layout, filters below products
- **Mobile (≤480px)**: Single-column grid, simplified navigation

### Layout Techniques
- **CSS Grid**: Product grid (auto-fit, minmax), testimonials, footer
- **Flexbox**: Header navigation, product cards, form layouts
- **CSS Variables**: Consistent theming (:root)
- **Media Queries**: Three responsive breakpoints
- **Animations**: Keyframe animations (fadeIn, pulse)
- **Transitions**: Hover effects and smooth interactions
